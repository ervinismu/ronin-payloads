# ronin-payloads-launch 1 "May 2022" Ronin "User Manuals"

## SYNOPSIS

`ronin-payloads launch` [*options*] {--file *FILE* \| *NAME*}

## DESCRIPTION

Launches a payload.

## ARGUMENTS

*NAME*
  The name of the payload to load.

## OPTIONS

`-f`, `--file` *FILE*
  Optionally loads the payload from the file.

`-h`, `--help`
  Print help information
