# ronin-payloads-console 1 "May 2022" Ronin "User Manuals"

## SYNOPSIS

`ronin-payloads console` [*options*]

## DESCRIPTION

Starts an interactive ruby console with `ronin-payloads` loaded.

## OPTIONS

`-h`, `--help`
  Print help information
